# Project Title

Pariwisata Sulawesi Selatan

## Description

Halaman web sederhana sebagai referensi mahasiswa di mata kuliah pemrograman web. 

## Getting Started

### Dependencies

* Web Server
* ex. Xampp, Appserc, etc

### Installing

* Sesuaikan variabel server ($dbhost, $dbuser, $dbpass) di halaman membuat_db.php dan membuat_tabel.php dengan data local server di perangkat masing-masing.
* Buat database dengan menjalankan halaman membuat_db.php dengan sebelumnya mengedit .
* Buat tabel dengan menjalankan halaman membuat_tabel.php
